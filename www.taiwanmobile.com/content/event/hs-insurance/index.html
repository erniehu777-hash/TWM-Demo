投保、取消問題

  • + 如何投保行動裝置保險？
    行動裝置保險僅限於台灣大哥大門市購買全新未拆封的行動裝置時才可以投保，離店則不可再投保。
  • + 如何取消行動裝置保險？
    請至台灣大哥大直營/加盟門市辦理。
  • + 我的舊手機有投保行動裝置保險，最近新買了新手機也加保保險，請問舊手機的保險是否會自動取消？
    1.不會，因為2張保單投保的行動裝置不同，您新手機的保單和舊手機的保單，是兩份不同的保險合約，故舊手機的保險會持續存在，直到保險期間到期。
    2.若要取消舊手機之保險，請至台灣大哥大直營/加盟門市辦理。
  • + 請問電信門號合約終止時(包括但不限於取消交易、退租、一退一租、被銷號、過戶、欠費拆機、攜碼移出至其他電信或有任何門號租用主體變更情形)，會影響我的行動裝置保險嗎？
    若您的電信門號合約有題目中任一情況，則行動裝置保險會一併終止。

    依保單條款約定，保險合約自然到期後十五個日曆天內，用戶仍可將投保的行動裝置拿到台灣大哥大門市送修，但行動裝置的損壞需發生在保險期間內，且是否理賠/如何理賠須由保險公司判定。

  • + 我目前的台灣大哥大小額帳單有欠費，會影響我的保險權益嗎？

    1. 用戶在取消保險後，富邦產險會寄發退費文件到用戶保單內記載的地址，用戶填寫完畢後需將退費文件寄回給保險公司，保險公司收到退費文件後，會依照短期費率退費給用戶(匯款)，退費時程約需2-3個月。
    2. 若用戶需更改更改保單內記載的地址，請向台灣大哥大門市、客服、富邦產險反映。
  • + 我投保的是新復原者預繳案，投保時在門市有當場預繳一筆保費，為什麼之後台灣大哥大帳單還有向我收費？
    您投保的行動裝置保險為月繳保單，而新復原者預繳案提供您於投保時，可先一次預繳前6/12/24個月保費的繳費方案，當預繳期間結束後，將恢復每月隨電信帳單收費的繳費模式，直到保險期間結束。
  • + 我投保的是新復原者月繳案，為什麼保險取消後，台灣大哥大電信帳單仍持續向我收取保險費？
    因台灣大哥大電信帳單出帳方式為「先使用後收費」，故用戶在取消保險前的費用，將會依照帳單週期陸續出帳讓用戶繳納(帳單週期根據每位用戶將有所不同)
  • + 我投保的是新復原者預繳案/月繳案，需要向公司報帳，請問哪裡可以取得繳款證明？
